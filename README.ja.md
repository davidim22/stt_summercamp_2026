# ⛪ 教会リアルタイム翻訳システム

韓国語の説教をリアルタイムで日本語に翻訳し、訪問者の携帯電話に表示するシステムです。

🌐 Languages: [한국어](README.md) | [日本語](README.ja.md) | [English](README.en.md)

---

## ✨ 機能

- 🎤 リアルタイム韓国語 → 日本語音声翻訳
- 📱 QRコードまたはPINで携帯電話から接続
- 🔐 6桁のPINでアクセス保護
- ⚡ 翻訳速度を3段階で調整可能（超高速 / 高速 / 正確）
- 🈂️ 漢字の上にひらがな（ふりがな）を表示する機能
- 🇰🇷 原文（韓国語）テキストも一緒に表示する機能
- 📜 テレプロンプター風の自動スクロール画面
- 🎙️ 様々な音声入力デバイスに対応（マイク、ミキサー、USBオーディオなど）
- 📋 リアルタイムシステムログの確認
- 🌐 同じWi-Fiでなくても接続可能（Cloudflare Tunnel使用）

---

## 📋 必要なもの

- Mac OS（SoxおよびLibreOffice関連機能はMac専用）
- [Node.js](https://nodejs.org)（v18以上）
- [OpenAI APIキー](https://platform.openai.com)
- Homebrew（Macのパッケージ管理ツール）

---

## 🚀 インストール方法

### 1. リポジトリをクローン

```bash
git clone https://github.com/davidim22/stt_summercamp_2026.git
cd stt_summercamp_2026
```

### 2. パッケージをインストール

```bash
npm install
```

### 3. 必要なプログラムをインストール

```bash
brew install sox
brew install cloudflare/cloudflare/cloudflared
```

### 4. 環境変数の設定

`.env` ファイルを作成し、APIキーを入力します：

```bash
nano .env
```

以下の内容を追加：

```
OPENAI_API_KEY=ここに自分のAPIキーを入力
```

保存：`Ctrl+X` → `Y` → `Enter`

> ⚠️ **重要**：`.env` ファイルは絶対にGitHubにアップロードしないでください。APIキーが漏洩すると、他人があなたのアカウントで料金を発生させる可能性があります。

### 5. サーバーを起動

```bash
node server.js
```

ブラウザが自動的に開き、`localhost:3000/control` ページが表示されます。

## 💻 使用方法

### 運営者（コントロールパネル）

1. サーバーを起動するとコントロールパネルが自動的に開きます
2. 🎙️ **オーディオ入力設定**で使用するマイクやミキサーを選択し、テストします
3. 📱 **QRコード**セクションで接続リンクとPINを確認します
4. 🎤 **翻訳コントロール**で**翻訳開始**を押すと翻訳が始まります（この時点からAPI費用が発生します）
5. 説教が終わったら**翻訳停止**を押して費用の発生を止めます

### 訪問者（携帯電話）

1. QRコードをスキャンするか、リンクを直接入力します
2. PIN番号を入力します（運営者にお尋ねください）
3. リアルタイムの日本語翻訳を確認します
4. 画面下部のボタンで文字サイズ、原文表示、ふりがな表示を調整できます

---

## ⌨️ コントロールパネルの機能

| ボタン | 機能 |
|---|---|
| ▶ 翻訳開始 | マイクと翻訳接続を開始（API費用発生） |
| ⏹ 翻訳停止 | マイクと翻訳接続を完全に停止（API費用停止） |
| 📢 字幕表示 | 携帯電話に字幕を表示（費用には影響なし） |
| 🔇 字幕非表示 | 携帯電話で字幕を非表示（費用には影響なし） |
| 🚀 超高速翻訳 | 文字単位で即座に表示（最速、やや粗く見える） |
| ⚡ 高速翻訳 | 単語単位で滑らかに表示（速く読みやすい） |
| 🎯 正確翻訳 | 完成した文単位で表示（最も正確、やや遅い） |
| 🔄 PIN再生成 | 新しい接続PINを生成（既存の接続者は切断されます） |

---

## 📱 携帯電話画面の機能

| ボタン | 機能 |
|---|---|
| A- / A / A+ | 文字サイズの調整（各デバイスに保存されます） |
| KO | 韓国語原文の表示オン/オフ |
| ふ | 漢字の上にふりがな（ひらがな）を表示オン/オフ |
| 🗑️ 全て削除 | 画面に蓄積された翻訳履歴を削除 |

すべての設定は各携帯電話に個別に保存されるため、複数人が同時に接続しても、それぞれ異なる設定を使用できます。

## 💰 API費用

OpenAI `gpt-realtime-translate` モデル基準：

```
$0.034 / 分

45分の説教 ≈ $1.53
3日間のキャンプ（1日2回）≈ $9.18
```

**翻訳開始**を押している間のみ費用が発生し、**翻訳停止**の状態では費用が発生しません（ただし、**字幕非表示**はマイク接続を維持するため費用が継続して発生します。費用を完全に止めるには翻訳停止を使用してください）。

支出制限の設定：[platform.openai.com](https://platform.openai.com) → Billing → Usage limits

## 🛠️ 技術スタック

- **バックエンド**：Node.js + Express
- **翻訳**：OpenAI `gpt-realtime-translate`（WebSocket）
- **リアルタイム配信**：Server-Sent Events（SSE）
- **音声キャプチャ**：Sox
- **ふりがな変換**：Kuroshiro + Kuromoji
- **外部接続**：Cloudflare Tunnel（ngrokをバックアップとして使用）
- **QRコード生成**：qrcodeパッケージ

---

## ⚠️ 注意事項

- `.env` ファイルは絶対にGitHubにアップロードしないでください
- 各ユーザーは自分のOpenAI APIキーが必要です
- Mac OSでのみ正常に動作します（Sox、system_profilerを使用）
- 安定したインターネット接続が必要です
- Cloudflare Tunnelの接続が失敗した場合、同じWi-Fi内でのみ接続可能です

## 🐛 トラブルシューティング

| 問題 | 解決方法 |
|---|---|
| ポート3000がすでに使用中 | `lsof -ti:3000 \| xargs kill -9` |
| Soxが見つからない | `brew install sox` |
| マイクの音量が低すぎる | オーディオ入力設定で別のデバイスを選択するか、マイクの音量を確認してください |
| 携帯電話が接続できない | PINが正しいか確認し、QRコードを再生成してみてください |
| 翻訳が日本語ではなく別の言語になる | APIキーを確認し、サーバーを再起動してみてください |
| APIキーエラー | `.env` ファイルに OPENAI_API_KEY が正しく入力されているか確認してください |
