# ⛪ 교회 실시간 번역 시스템

한국어 설교를 일본어로 실시간 번역하여 방문자의 휴대폰으로 보여주는 시스템입니다.

🌐 Languages: [한국어](README.md) | [日本語](README.ja.md) | [English](README.en.md)

---

## ✨ 기능

- 🎤 실시간 한국어 → 일본어 음성 번역
- 📱 QR 코드 또는 PIN으로 휴대폰 접속
- 🔐 6자리 PIN으로 접속 보호
- ⚡ 번역 속도 3단계 조절 (초고속 / 빠른 번역 / 정확한 번역)
- 🈂️ 한자 위 히라가나(후리가나) 표시 기능
- 🇰🇷 원문(한국어) 텍스트 함께 보기 기능
- 📜 텔레프롬프터 스타일 자동 스크롤 화면
- 🎙️ 다양한 오디오 입력 장치 지원 (마이크, 믹서, USB 오디오 등)
- 📋 실시간 시스템 로그 모니터링
- 🌐 같은 와이파이가 아니어도 접속 가능 (Cloudflare Tunnel)

---

## 📋 필요한 것

- Mac OS (Sox와 LibreOffice 관련 기능은 Mac 전용)
- [Node.js](https://nodejs.org) (v18 이상)
- [OpenAI API 키](https://platform.openai.com)
- Homebrew (Mac 패키지 관리자)

---

## 🚀 설치 방법

### 1. 저장소 복제

```bash
git clone https://github.com/davidim22/stt_summercamp_2026.git
cd stt_summercamp_2026
```

### 2. 패키지 설치

```bash
npm install
```

### 3. 필요한 프로그램 설치

```bash
brew install sox
brew install cloudflare/cloudflare/cloudflared
```

### 4. 환경 변수 설정

`.env` 파일을 생성하고 API 키를 입력합니다:

```bash
nano .env
```

다음 내용을 추가:

```
OPENAI_API_KEY=여기에_본인의_API_키_입력
```

저장: `Ctrl+X` → `Y` → `Enter`

> ⚠️ **중요**: `.env` 파일은 절대 GitHub에 올리지 마세요. API 키가 노출되면 다른 사람이 비용을 사용할 수 있습니다.

### 5. 서버 실행

```bash
node server.js
```

브라우저가 자동으로 열리며 `localhost:3000/control` 페이지가 표시됩니다.

## 💻 사용 방법

### 운영자 (컨트롤 패널)

1. 서버를 실행하면 컨트롤 패널이 자동으로 열립니다
2. 🎙️ **오디오 입력 설정**에서 사용할 마이크나 믹서를 선택하고 테스트합니다
3. 📱 **QR 코드** 섹션에서 접속 링크와 PIN을 확인합니다
4. 🎤 **번역 컨트롤**에서 **번역 시작**을 누르면 번역이 시작됩니다 (이때부터 API 비용이 발생합니다)
5. 설교가 끝나면 **번역 중지**를 눌러 비용 발생을 멈춥니다

### 방문자 (휴대폰)

1. QR 코드를 스캔하거나 링크를 직접 입력합니다
2. PIN 번호를 입력합니다 (운영자에게 문의)
3. 실시간 일본어 번역을 확인합니다
4. 화면 하단 버튼으로 글자 크기, 원문 보기, 후리가나 표시를 조절할 수 있습니다

## ⌨️ 컨트롤 패널 기능

| 버튼 | 기능 |
|---|---|
| ▶ 번역 시작 | 마이크와 번역 연결 시작 (API 비용 발생) |
| ⏹ 번역 중지 | 마이크와 번역 연결 완전히 중지 (API 비용 중지) |
| 📢 자막 보이기 | 휴대폰에 자막 표시 (비용에 영향 없음) |
| 🔇 자막 숨기기 | 휴대폰에서 자막 숨김 (비용에 영향 없음) |
| 🚀 초고속 번역 | 글자 단위로 즉시 표시 (가장 빠름, 다소 거칠게 보임) |
| ⚡ 빠른 번역 | 단어 단위로 부드럽게 표시 (빠르면서 읍기 편함) |
| 🎯 정확한 번역 | 완성된 문장 단위로 표시 (가장 정확, 다소 느림) |
| 🔄 PIN 재생성 | 새로운 접속 PIN 생성 (기존 접속자는 끊김) |

## 📱 휴대폰 화면 기능

| 버튼 | 기능 |
|---|---|
| A- / A / A+ | 글자 크기 조절 (각 기기에 저장됨) |
| KO | 한국어 원문 표시 켜기/끄기 |
| ふ | 한자 위 후리가나(히라가나) 표시 켜기/끄기 |
| 🗑️ 전체 지우기 | 화면에 누적된 번역 기록 삭제 |

모든 설정은 각 휴대폰에 개별적으로 저장되어, 여러 명이 동시에 접속해도 서로 다른 설정을 사용할 수 있습니다.

## 💰 API 비용

OpenAI `gpt-realtime-translate` 모델 기준:

```
$0.034 / 분

45분 설교 ≈ $1.53
3일 캠프 (하루 2회) ≈ $9.18
```

**번역 시작**을 누른 동안만 비용이 발생하며, **번역 중지** 또는 **자막 숨기기** 상태에서는 비용이 발생하지 않습니다 (단, 자막 숨기기는 마이크 연결을 유지하므로 비용이 계속 발생합니다 — 비용을 완전히 멈추려면 번역 중지를 사용하세요).

지출 한도 설정: [platform.openai.com](https://platform.openai.com) → Billing → Usage limits

## 🛠️ 기술 스택

- **백엔드**: Node.js + Express
- **번역**: OpenAI `gpt-realtime-translate` (WebSocket)
- **실시간 전송**: Server-Sent Events (SSE)
- **오디오 캡처**: Sox
- **후리가나 변환**: Kuroshiro + Kuromoji
- **외부 접속**: Cloudflare Tunnel (ngrok 백업)
- **QR 코드 생성**: qrcode 패키지

---

## ⚠️ 주의사항

- `.env` 파일을 절대 GitHub에 업로드하지 마세요
- 각 사용자는 본인의 OpenAI API 키가 필요합니다
- Mac OS에서만 정상 작동합니다 (Sox, system_profiler 사용)
- 안정적인 인터넷 연결이 필요합니다
- Cloudflare Tunnel 연결이 실패하면 같은 와이파이에서만 접속 가능합니다

## 🐛 문제 해결

| 문제 | 해결 방법 |
|---|---|
| 포트 3000이 이미 사용 중 | `lsof -ti:3000 \| xargs kill -9` |
| Sox를 찾을 수 없음 | `brew install sox` |
| 마이크 음량이 너무 낮음 | 오디오 입력 설정에서 다른 장치를 선택하거나 마이크 볼륨을 확인하세요 |
| 휴대폰이 연결되지 않음 | PIN이 올바른지 확인하고, QR 코드를 새로고침해보세요 |
| 번역이 일본어가 아닌 다른 언어로 나옴 | API 키와 서버를 재시작해보세요 |
| API 키 오류 | `.env` 파일에 OPENAI_API_KEY가 올바르게 입력되었는지 확인하세요 |
